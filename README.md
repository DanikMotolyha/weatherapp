### Getting Started

1. Clone the repo
```sh
git clone https://github.com/DanikMotolyha/weatherapp.git
```
2. go to the project
```sh
cd weatherapp
```
3. use gradle to start Spring Boot Application
```sh
gradle build
```
4. use docker-compose to run application with postgres
```sh
docker-compose up
```
